# Samarth Foundation Old Age Home Website

## Meeting Summary

- Date: 3 July 2026
- Mode: Online
- Purpose: Review the old website, review the demo website, and define improvements for the final website.

## What Was Agreed

The current old-age-home website was described as outdated and limited in functionality. The demo website was reviewed as a better starting point, and the group agreed to improve it further before building the final version.

### Main Improvement Themes

- Modern UI and UX
- Mobile responsiveness
- Accessibility and readability
- Multi-language support
- Resident stories and short videos
- Login system
- Improved statistics and journey sections
- Volunteer registration form
- Contact form
- Admin panel for content management
- Backend integration for donations and website management
- Scalable architecture
